Network Architecture

An overview of how the master system communicates with relay bots and forwards telemetry data to cloud services for monitoring.

Communication Flow

The system uses a centralized Master Arduino (UNO R4 WiFi) to control multiple relay bots via HC-12 wireless modules. Telemetry data collected from each bot is transmitted back to the master and forwarded to a cloud server using HTTP JSON POST requests.

The backend server, hosted on Render with a Flask application, stores telemetry data in a database. This data is then accessed by a web-based dashboard for real-time monitoring and analysis.
